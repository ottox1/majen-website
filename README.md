# Majen - Your Hub for African Fashion

A landing page for Majen, a fashion marketplace connecting African designers with customers worldwide.

## Features

- Responsive design for all devices
- Interactive FAQ accordion
- Smooth scroll navigation
- Mobile-friendly hamburger menu
- Scroll-triggered animations

## Project Structure

```
MAJEN/
├── index.html      # Main HTML file
├── styles.css      # Stylesheet
├── script.js       # JavaScript interactions
└── README.md       # This file
```

## Getting Started

### Local Development

Simply open `index.html` in your browser, or use a local server:

```bash
# Using Python
python3 -m http.server 8000

# Using Node.js
npx serve
```

Then visit `http://localhost:8000` (Python) or `http://localhost:3000` (serve).

## Customization

### Adding Images

Replace placeholder divs with actual images:

1. Add your images to an `images/` folder
2. Update the HTML to use `<img>` tags or CSS `background-image`

### Colors

Edit CSS variables in `styles.css`:

```css
:root {
    --primary-color: #1a1a6e;
    --primary-dark: #0f0f4a;
    --text-dark: #1a1a1a;
    --text-light: #666666;
}
```
